## Product Name

**Kazadot AI**

---

## Product Description

Kazadot AI is a predictive analysis tool that analyzes historical accident data on the Turkish highway network (TURKSTAT 2025 data) alongside open municipal data to preemptively highlight high-risk road segments for drivers, courier networks, and city administrations. Unlike standard navigation applications that merely display current traffic, this project calculates the instantaneous "black spot" accident risk using machine learning (XGBoost). Furthermore, through integrated Artificial Intelligence Agents (AI Agents), it interprets risk analyses using natural language processing and provides autonomous intervention recommendations to municipal and fleet managers.

---

## Product Features

### Spatio-Temporal Risk Prediction

Calculating instantaneous risk scores for road segments using open data (Izmir Municipality & TURKSTAT) and real-time weather data.

### Smart Route Recommendation

Presenting users with "safe routes" that have the lowest accident risk via an interactive map (MapLibre).

### Autonomous Reporting Agent (LLM Agent)

Detecting sudden spikes in risk rates and generating actionable warning alerts with root causes for municipal/fleet managers.

Example:

> "80% risk increase due to heavy precipitation"

### Executive Dashboard

A web interface where municipalities can track and prioritize the top 20 accident black spots requiring immediate intervention.
